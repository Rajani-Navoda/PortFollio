Simple Chess Game

This project is a basic implementation of a chess game in Java, designed to follow standard chess rules and provide a graphical user interface (GUI) for easy interaction. It includes movement logic for all chess pieces, such as the Queen, King, Rook, Bishop, Knight, and Pawn. This project serves as a learning exercise in Java programming, object-oriented design, and GUI development. Key features include basic chessboard functionality, enforcement of valid moves for all pieces, and a focus on essential chess mechanics (excluding check and checkmate).

Simple Java Chat Application

This project contains the development of a basic chat application, focusing on backend functionality. The application features both server and client components, allowing multiple clients to connect to the server and communicate with each other.
